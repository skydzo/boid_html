SCENARIOS MANAGER

This core feature allows creation and management of complex scenarios, starting with a spreadsheet for non-programmer users. We can use this tool as the conductor of a game. This feature was done on Unreal Engine 4 in C++. It's accessible everywhere in blueprints as in C++ with main functions, in interfaces and fully network replicated. This tool convert the spreadsheet to JSON to an object that we can see as a tree. This tree represent a series of steps that users must complete. You are able to lock the user in the tree, to allows him to skip steps or adding some errors. You can also add visual effects on objects according to the steps. At the end of the experience, you can retrieve all datas like errors or just informations about the steps. Datas are saved as a JSON and linked to the desktop web solution

VR TOOLS

This class cover a bunch of really precise VR interactions embedded in for surgical uses. This feature was done with Unreal Engine 4 with only Blueprints. Thanks to this system you can create a tool with part of other tools. You can manage the insertion and extractions, slider movement, screwing, different types of rotations, a tilt movement or all of them at same time. The class is fully replicated and optimized for networking. It's linked with the experience management system to manage tools according to the steps. You can choose to lock the user or let him play with all the real possibilities. Finally you can choose to add a undo system to go back where you want in the manipulation.

Plugin Networking UX VR

STACKABLE

This tool allows to stack items in a custom container. This feature was made on Unreal Engine 4 with mainly C++, optimized for Oculus Quest usage. You can stack any object in any container defined by a size of a 3 dimensional grid and a rotation rule. The program will search for a solution and stack the object if found.

Plugin C++ 3D Grid VR

UNIVRSTORE

UnivrStore is a web based desktop solution for VR application management. It allows you to manage the users (create organizations, adding user's roles and permissions), localization (add languages and manage translations) and provide a licensing system. You can upload, manage and update your VR application. It also handle sessions creation and a browsing management system. The application can receive automatic update from the server in order to download the last update of the client and can also work offline with cached memory management.
